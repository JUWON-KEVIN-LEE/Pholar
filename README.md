# PHOLAR

## 소개
안드로이드 개발 ( with Firebase ) 공부를 목적으로 네이버 소셜 포토블로그 앱인 'PHOLAR' 를 카피하기로 했습니다.</br></br>
일주일이라는 짧은 기간 동안 개발과 공부를 병행해야 했기 때문에 특수한 기능들을 빼고, 기존에 학습했던 기능들과
배워보고 싶고, 사용해보고 싶은 기능들을 위주로 구현하였습니다.</br>
